# netflix-data-analysis
Exploratory Data Analysis on 8000+ Netflix titles using Python, Pandas, Matplotlib and Seaborn
 🎬 Netflix Data Analysis

 📌 Project Overview
Performed Exploratory Data Analysis (EDA) on a dataset of 8,000+ Netflix 
titles to uncover content trends, genre distributions, country-wise 
production, and yearly growth patterns.

 🛠️ Tools & Libraries Used
- Python
- Pandas (data cleaning & analysis)
- Matplotlib (visualizations)
- Seaborn (styled charts)
- Jupyter Notebook

 📊 Key Findings
- 69% of Netflix content is Movies, 31% is TV Shows
- USA produces the highest number of titles globally
- Netflix content grew significantly between 2015–2019
- International Movies is the most common genre
- TV-MA is the most frequent content rating

 📁 Project Structure
- `netflix_analysis.ipynb` — Main analysis notebook
- `netflix_titles.csv` — Raw dataset
- `movies_vs_shows.png` — Movies vs TV Shows pie chart
- `top_countries.png` — Top 10 countries bar chart
- `content_per_year.png` — Content growth line chart
- `top_genres.png` — Top 10 genres bar chart
- `ratings.png` — Ratings distribution chart
